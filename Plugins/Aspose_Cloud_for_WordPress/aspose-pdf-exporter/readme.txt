Aspose PDF Exporter exports posts into a Pdf file. Currently this plugin works as a bulk action of post / pages or of any custom post type.

== Description ==

Aspose PDF Exporter Plugin for WordPress allows administrators to export post contents to PDF document. This module demonstrates very powerful Export feature provided by Aspose.Pdf. It adds a simple bulk action in the drop down "Aspose Export to PDF" and as soon as the bulk action is called, it exports all posts to a PDF document.

== Installation ==

1. Creating an account at http://cloud.aspose.com is a prerequisite for adding Aspose Key & ID in the settings. Please check the link to see how you can get app keys http://www.aspose.com/docs/display/rest/Creating+a+New+App+and+Getting+App+Key
2. Download the plugin and extract it.
3. Upload the directory '/aspose-pdf-exporter/' to the '/wp-content/plugins/' directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Click on 'Aspose Pdf Exporter' link under Settings menu to access the admin section.

== Frequently Asked Questions ==

= Can I use Aspose PDF Exporter plugin without having account on cloud.aspose.com ? =
No, you cannot use this plugin without having account on aspose cloud. You need to go https://cloud.aspose.com/SignUp and create a new account with aspose cloud.
= From where I can get App SID & App Key ? =
Login with your aspose account by going to the link https://cloud.aspose.com/SignIn . Then go the My Apps section and check the Manage My Apps section, you will find your credentials. Put these on the “Aspose Doc Exporter” plugin settings page.
= Can I export posts content to Doc file? =
No, this plugin is only for exporting posts into pdf files. For exporting posts into doc file , you have to use our different plugin, which is Aspose Doc Exporter Plugin.
